# Admitly Mobile App

React Native mobile app built with Expo.

## Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm dev

# Run on iOS
pnpm ios

# Run on Android
pnpm android
```

## Building

```bash
# Build for Android
pnpm build:android

# Build for iOS
pnpm build:ios
```

## Structure

- `app/` - Expo Router pages
- `components/` - React Native components
- `lib/` - Utilities and configurations
